# Changelog

## v1.0.1

**「壊れている」問題と、ファイル選択が効かない不具合を修正しました。**

### 修正

- **「壊れているため開けません」への対処を同梱** — 配布zipに `初回セットアップ.command` を同梱しました。ダウンロード後にこれを実行（右クリック →「開く」）すると、未署名アプリの検疫属性（`com.apple.quarantine`）を自動で取り除き、アプリが起動できるようになります。
- **「動画を選択」やドラッグ&ドロップが反応しない不具合を修正** — ネイティブ判定（`window.pywebview.api`）をスクリプト読み込み時に固定していたため、アプリ版でもブラウザ扱いになりファイル選択ダイアログが開かないことがありました。呼び出し時に毎回判定するよう修正しました。

### 使い方（アップデート）

1. リリースから `AutoTelop-mac.zip` をダウンロードして展開
2. `AutoTelop.app` をアプリケーションフォルダへ移動
3. 同梱の `初回セットアップ.command` を**右クリック →「開く」**で実行
4. `AutoTelop.app` を起動 → 「動画を選択」またはドラッグ&ドロップで動画を追加

## v1.0.0

**Auto Telop が Mac用のGUIアプリになりました。** ターミナルもブラウザも不要です。

### 新機能

- **GUIローカルアプリ（PyWebView）** — アプリを起動して動画を扱えます。既存の校閲エディタをアプリ内に内蔵。
- **ライブラリ画面** — 動画をドラッグ&ドロップ（複数可）または「＋ 動画を選択」で追加。
  - 抽出が終わると、動画と文字起こし結果をまとめた**エントリー**が一覧に追加されます。
  - エントリーをクリックすると校閲画面に移動します。
  - 複数動画は**1本ずつ順番に**処理され、進捗が表示されます。
  - アプリを閉じて再度開いても、**以前のエントリーはそのまま残ります**（`projects/` に保存、DB不要）。
  - エントリーの削除に対応。
- **保存先の指定** — 保存時にネイティブのダイアログで**出力先フォルダ・ファイル名を選べます**。前回選んだフォルダを記憶します。
- **バージョン管理とアプリ内アップデート** — 「更新を確認」ボタンから GitHub Releases の最新版を確認し、ダウンロードできます。バージョンは `VERSION` で管理。

### 開発・配布

- `desktop.py` … PyWebViewランチャー（Flaskを内部起動＋ネイティブダイアログ連携）
- `jobs.py` … 抽出ジョブの順次処理キュー
- `updater.py` / `version.py` … 更新確認・ダウンロード
- `AutoTelop.spec` / `build_macos.sh` … PyInstallerによる `.app` ビルド
- README・セットアップガイド（`guide.html`）をGUIアプリ向けに刷新

### 補足

- Whisperのモデルは初回起動時に自動ダウンロードされます（約1.5GB）。
- FFmpeg（`brew install ffmpeg`）が必要です。アプリには同梱されません。
- クロスプラットフォーム構成のため、Windows版（`.exe`）は今後追加しやすくなっています。
