Auto Telop
セットアップガイド

動画を渡すだけで自動テロップ。Mac用のGUIアプリで、Whisper文字起こし → 校閲 → Final Cut Pro用のFCPXMLを出力します。ターミナルもブラウザも不要です。

STEP 2

必要なもの

  • Mac（macOS 12以降を推奨）
  • FFmpeg（動画の解析に使います。brew install ffmpeg）
  • Final Cut Pro（FCPXMLのインポート先）
  • ディスク空き容量 3GB以上（Whisperモデル用）
Whisperのモデルはアプリの初回起動時に自動でダウンロードされます（約1.5GB、~/.cache/whisper）。
アプリを自分でビルドする場合は、追加で Git と Python 3.9以上 が必要です（次のステップ）。
STEP 3

アプリを準備する

配布された AutoTelop.app がある場合は、アプリケーションフォルダなどに置くだけでOK。ここでは自分でビルドする手順を説明します。

Homebrewをインストール（まだの場合）

# Homebrewが入っているか確認
$ brew --version

# 入っていなければインストール（パスワードを聞かれます）
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Git, Python, FFmpeg をインストール

$ brew install git python ffmpeg

ダウンロードしてアプリをビルド

$ cd ~
$ git clone https://github.com/kazuu234/auto-telop.git
$ cd auto-telop
$ ./build_macos.sh

...（依存インストール & ビルド。数分かかります）...
Done. App is at: dist/AutoTelop.app

dist/AutoTelop.app が生成されます。ダブルクリックで起動できます。

「壊れているため開けません」と出たら： 署名していないアプリのため、ダウンロード直後は macOS の検疫属性で起動がブロックされます。配布zipに同梱の 「初回セットアップ.command」をダブルクリックすると自動で解除され、以降は普通にダブルクリックで起動できます（初回のみ）。
ビルドせずに試したいときは python desktop.py でも同じGUIアプリが起動します（先に python3 -m venv venv && source venv/bin/activate && pip install -r requirements.txt が必要）。
STEP 4

動画を追加してテロップ抽出

AutoTelop.app を起動すると、ライブラリ画面が開きます。

① 動画を追加

ウィンドウに動画をドラッグ&ドロップ（複数まとめて可）。または「＋ 動画を選択」をクリック。

② 自動で抽出

Whisperによる文字起こしが自動で始まります。複数動画は1本ずつ順番に処理され、進捗が表示されます。

③ エントリーが追加

抽出が終わると、動画と文字起こし結果をセットにしたエントリーが一覧に並びます。

④ いつでも続けられる

抽出中でも別の動画を追加できます。アプリを閉じても、次に開いたときに一覧はそのまま残ります。

処理時間の目安：18分の動画で約5〜15分（PCスペックにより変動）。GPUがあるMacなら速くなります。
STEP 5

校閲する

ライブラリでエントリーをクリックすると、校閲画面が開きます。

左：動画プレーヤー

動画を再生しながら、対応するテロップ行が自動ハイライトされます。

右：テロップ一覧

各行のテキストを直接編集できます。タイムスタンプ・CPS（文字/秒）も確認できます。

行をクリック

その位置に動画がジャンプします。テキスト欄をクリックすれば直接編集。

行のマージ

Shift+クリックで複数行を選択 → 「▶ マージ」ボタンで結合できます。

主な校閲作業

  • Whisperの誤変換を修正する（例: monograph → モノグラフ）
  • 短すぎる行をマージする
  • 検索機能で特定の単語をまとめて確認する
  • CPS（文字/秒）が高すぎる行は文を短くする

左上の 「← ライブラリ」 でいつでも一覧に戻れます。校閲が終わったら 「💾 保存」ボタン を押してください。

STEP 6

保存してFCPにインポート

「💾 保存」を押すと、保存先を選ぶダイアログが表示されます。フォルダとファイル名を指定して保存してください。次回は前回選んだフォルダが初期値になります。

# 例：選んだ保存先に出力される
~/Movies/買ってよかった_テロップ付き.fcpxml

出力したFCPXMLファイルをFinal Cut Proで開く手順：

  1. Final Cut Pro を開く
  2. メニュー → ファイル → 読み込む → XML...
  3. 保存したFCPXMLファイルを選択
  4. タイムラインにテロップ付き動画が読み込まれる
完了！ FCP上でテロップの位置やスタイルをさらに微調整できます。
STEP 7

設定をカスタマイズする

テロップのフォント・サイズ・位置などは、校閲画面の「テロップスタイル プレビュー」から変更・保存できます。より細かい設定は config.yaml を直接編集します（初回起動時に自動生成されます）。

# テロップの見た目
style:
  font_name: "A P-OTF A1Gothic StdN"
  font_size: 35
  bold: true
  outline_width: 3

# Whisperの精度（large にすると精度↑ だが遅い）
whisper:
  model: "medium"   # tiny / base / small / medium / large
  language: "ja"

# セグメント分割の調整
segment:
  max_chars: 25     # 1行の最大文字数
  max_duration: 4.0 # 1セグメントの最大秒数
STEP 8

アプリを更新する

ライブラリ画面の右上「更新を確認」ボタンを押すと、GitHub Releasesの最新版と現在のバージョンを比べます。

  • 新しいバージョンがあると、画面上部に案内が表示されます。
  • 「ダウンロード」を押すとダウンロードフォルダに新しいアプリが保存されます。
  • ダウンロードした新しいアプリと、いまのアプリを差し替えてください。
現在のバージョンは、ライブラリ画面の左上（アプリ名の横）に v1.0.0 のように表示されます。
STEP 9

よくあるトラブル

アプリが「壊れているため開けません」と表示される

署名していないアプリのため、macOSの検疫属性で起動がブロックされている状態です。以下の手順で解消できます。

  1. ダウンロードした AutoTelop-mac.zip を展開する
  2. AutoTelop.app をアプリケーションフォルダ（または任意の場所）へ移動する
  3. 同じフォルダにある 「初回セットアップ.command」をダブルクリックする
  4. ターミナルに [OK] 完了しました！ と表示されたら、AutoTelop.app をダブルクリックで起動できます

「初回セットアップ.command」は、検疫属性を取り除く xattr -cr を実行するだけの安全なスクリプトです。

ffprobe: command not found と出る／抽出がすぐ失敗する

FFmpegが入っていません。brew install ffmpeg を実行してください。

テロップ抽出が非常に遅い

config.yaml の whisper.model を "small" や "base" に変えると速くなります（精度は少し落ちます）。

校閲画面で動画が再生されない

mp4形式以外の動画（.mov, .avi等）の場合、再生できないことがあります。FFmpegでmp4に変換してから使ってください。

$ ffmpeg -i input.mov -c:v libx264 -c:a aac output.mp4

（ビルド時）command not found: python と出る

python3 に読み替えて実行してください。pip も pip3 に変えてください。
