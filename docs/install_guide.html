AutoTelop / Setup guide

ダウンロードしたアプリを開けるようにする

署名していないアプリのため、初回はmacOSにブロックされます。同梱の「初回セットアップ.command」を1回だけ実行すれば、以降はダブルクリックで起動できます。

⚠ 注意：途中で出るダイアログの「ゴミ箱に入れる」は押さないでください。押すとセットアップ用のファイルが消えてしまいます。迷ったら「完了」を選んでください。
STEP 01

ダウンロードした zip を確認

ダウンロードフォルダに AutoTelop-mac.zip があることを確認し、ダブルクリックして展開します。

STEP 02

AutoTelop をアプリケーションフォルダへ

展開してできた AutoTelop-mac フォルダの中に、AutoTelop（アプリ本体）と 初回セットアップ.command が入っています。AutoTelop を「アプリケーション」フォルダへドラッグ&ドロップしてください。

STEP 03

古いバージョンがあれば置き換える

以前のバージョンが入っている場合は確認ダイアログが出ます。「置き換える」を選んでください。（初めてのインストールならこの画面は出ません）

STEP 04

初回セットアップ.command を実行

展開したフォルダに戻り、初回セットアップ.command をダブルクリックします。「開けません」という警告が出ますが、想定どおりの動作です。次のステップに進んでください。

STEP 05

警告ダイアログを閉じる

「"初回セットアップ.command"は開いていません」という警告が出たら、「完了」を押して閉じます。（「ゴミ箱に入れる」は押さないでください）

STEP 06

システム設定で許可する

システム設定 → プライバシーとセキュリティ を開き、下の方にスクロールします。「"初回セットアップ.command"がブロックされました」という表示の横にある「このまま開く」を押します。

STEP 07

確認ダイアログでもう一度許可

確認のダイアログが表示されるので、「このまま開く」を選びます。

STEP 08

パスワードで承認

Touch ID またはパスワードでの承認画面が出ます。「パスワードを使用...」からMacのログインパスワードを入力してください。

STEP 09

完了メッセージを確認

ターミナルが開いて処理が始まります。「[OK] 完了しました！」と表示されたら成功です。このウィンドウは閉じて構いません。

macOS 14以前を使っている場合 STEP 04〜08の代わりに、「初回セットアップ.command」を右クリック →「開く」→「開く」だけで実行できます。
Done

AutoTelop.app をダブルクリックすれば起動します。次回からはこの手順は不要です。
